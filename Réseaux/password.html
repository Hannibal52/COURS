Récupération de mot de passe pour un switch Cisco

La procédure de récupération de mot de passe varie en fonction du modèle et du fabricant de votre switch. Cependant, voici les étapes générales pour la récupération de mot de passe d'un switch Cisco :

  • Connectez un câble de console série de votre ordinateur au port de console du switch à l'aide d'un adaptateur approprié.
  • Ouvrez un émulateur de terminal sur votre ordinateur et configurez-le pour utiliser le port de console série avec les paramètres de ligne suivants : 9600 bauds, 8 bits de données, pas de parité et 1 bit d'arrêt.
  • Allumez le switch. Lorsque le switch démarre, appuyez sur la touche Ctrl+Break pour entrer en mode ROM Monitor (ROMmon).
  • Dans le mode ROMmon, tapez la commande "confreg 0x2142" pour modifier la configuration du switch pour qu'il ignore la configuration de démarrage au prochain redémarrage.
  • Redémarrez le switch à l'aide de la commande "reset". Le switch démarre en ignorant la configuration de démarrage.
  • Tapez la commande "enable" pour accéder au mode d'administration du switch.
  • Tapez la commande "copy startup-config running-config" pour copier la configuration de sauvegarde dans la mémoire vive (RAM).
  • Tapez la commande "configure terminal" pour accéder au mode de configuration.
  • Tapez la commande "no enable secret" pour supprimer le mot de passe d'administration.
  • Tapez la commande "config-register 0x2102" pour réinitialiser la configuration de démarrage.
  • Tapez la commande "write memory" pour enregistrer les modifications dans la configuration de démarrage.
  • Redémarrez le switch à l'aide de la commande "reload". Le switch démarre avec la configuration de démarrage par défaut et sans mot de passe d'administration.

Il est important de noter que cette procédure effacera tous les mots de passe configurés sur le switch et réinitialisera la configuration de démarrage. Par conséquent, vous devrez reconfigurer le switch selon vos besoins après avoir récupéré le mot de passe.
