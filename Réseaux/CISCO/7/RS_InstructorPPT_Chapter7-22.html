R3 :



Envoie les dernières 
informations relatives au réseau 
10. 4. 0. 0 via l'interface Serial 
0/0/1.



Envoie les dernières 
informations relatives aux 
réseaux 10. 2. 0. 0 et 10. 3. 0. 0 
via l'interface FastEthernet 0/0.



Reçoit les dernières 
informations envoyées par R2 
sur le réseau 10. 1. 0. 0 avec 
une métrique égale à 2.



Stocke le réseau 10. 1. 0. 0 
dans la table de routage avec 
une métrique égale à 2.



Une mise à jour identique 
depuis R2 contient des 
informations sur le réseau 10. 2. 
0. 0 avec une métrique égale 
à 1. Aucune modification n'est 
intervenue ; par conséquent, les 
informations de routage restent 
les mêmes.
