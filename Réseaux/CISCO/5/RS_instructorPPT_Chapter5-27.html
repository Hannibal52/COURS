

Par défaut, une interface SVI est créée pour le VLAN par défaut 
(VLAN1). Cela rend possible l'administration à distance du 
commutateur.



Toutes les interfaces SVI supplémentaires doivent être créées par 
l'administrateur.



Elles sont créées la première fois que le mode de configuration 
d'interface VLAN est utilisé pour une interface SVI VLAN particulière.



La commande 

interface vlan 10

utilisée pour la première fois crée 

une interface SVI appelée VLAN 10.



Le numéro de VLAN indiqué correspond à l'étiquette de VLAN 
associée aux trames de données d'un trunk encapsulé 802.1Q.



Pour chaque création d'interface SVI, assurez-vous que le VLAN 
correspondant est présent dans la base de données des VLAN.
