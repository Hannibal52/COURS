Le gestionnaire de base de données des commutateurs 
(SDM, Switch Database Manager) de Cisco fournit 
plusieurs modèles pour le commutateur 2960.



Le modèle SDM

lanbase-routing

peut être activé pour que 

le commutateur puisse acheminer le trafic entre les VLAN 
et prenne en charge le routage statique.



La commande 

show sdm prefer

permet de savoir quel 

modèle est utilisé.



Pour changer le modèle SDM, passez en mode de 
configuration globale et utilisez la commande

sdm prefer

.
