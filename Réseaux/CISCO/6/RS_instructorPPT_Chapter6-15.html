Configuration des routes statiques IPv4

Options de tronçon suivant

Le tronçon suivant peut être identifié par une adresse IP, 
une interface de sortie, ou les deux. La manière dont la 
destination est spécifiée crée un des trois types de route 
suivants :



Route de tronçon suivant : seule l'adresse IP du 
tronçon suivant est spécifiée.



Route statique connectée directement : seule l'interface 
de sortie du routeur est spécifiée.



Route statique entièrement spécifiée : l'adresse IP de 
tronçon suivant et l'interface de sortie sont spécifiées.
