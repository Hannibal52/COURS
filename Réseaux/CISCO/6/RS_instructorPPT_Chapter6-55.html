Chapitre 6 : résumé



Les routes statiques peuvent être configurées avec une 
adresse IP de tronçon suivant, communément l'adresse IP du 
routeur de tronçon suivant. 



En cas d'utilisation d'une adresse IP de tronçon suivant, le 
processus de la table de routage doit convertir cette adresse 
en interface de sortie. 



Pour les liens série point à point, il est généralement plus 
efficace de configurer une interface de sortie sur la route 
statique. 



Sur les réseaux à accès multiple comme Ethernet, une 
adresse IP de tronçon suivant et une interface de sortie 
peuvent être configurées sur la route statique.



Les routes statiques ont une distance administrative par défaut 
de « 1 ».
