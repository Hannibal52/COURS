ACL Routeur cisco

  1. Définit le temps d'inactivité avant que la session ne se termine :

    2. R1(config-line)#exec-timeout 0 60

  2. Affiche les sessions en cours SSH :

    3. sh ssh

  3. Définit le nombre de tentatives autorisées pour une connexion SSH :

    4. ip ssh authentication-retries ?

  4. Définit le temps d'attente pour négocier une clé de session et une méthode de chiffrement avec un client SSH :

    5. ip ssh time-out 60

  5. Affiche la configuration SSH :

    6. Show ip ssh

  6. Modifie les clés SSH :

    7. crypto key ?

  7. Force une longueur minimale pour les mots de passe :

    8. security passwords min-length 10

  8. Bloque les connexions pendant 10 secondes si 3 tentatives de connexion échouent dans les 30 secondes :

    9. login block-for 10 attempts 3 within 30

  9. Affiche les paramètres de connexion :

    10. show login

  10. Bloque les messages de connexion pour les clients ne figurant pas sur la liste d'accès standard "adminacl" :

    11. login quiet-mode access-class adminacl

  11. Gestion des privilèges :
    • Définit un mot de passe secret et un niveau d'accès :

      • lenable secret level X ciscoX

    • Définit les privilèges de niveau d'accès pour les utilisateurs :

      • lprivilege exec,configure,interface level X
