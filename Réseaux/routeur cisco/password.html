Cisco Router Passwords

Passwords are used to restrict access to a CISCO router. Passwords can be given to the virtual terminal lines and the console line. Moreover, password can be set privileged EXEC mode. This page will provide the information to set up CISCO router password to virtual terminal lines, console lines and privileged mode. Even more, you will learn how to encrypt the password also.

Configuring Console Password

Console password is useful on a network on which multiple people have to access to the router. The persons who are not authorized, can't access the router. Thus it prevents unauthorized person from accessing the router.

To configure the console password of a CISCO router:

Router>enable
Router#config t
Router(config)#line console 0
Router(config-line)#password cisco 
Router(config-line)#login 
Router(config-line)#

Configuring Virtual Terminal Password

Virtual terminal password refers to telnet password. Through telnet, you can access the router and can change anything. So, it's really important to protect the use of telnet by giving a strong password.

To configure the virtual terminal password of a CISCO router:

Router>enable 
Router#configure terminal 
Router(config)#line vty 0 4 
Router(config-line)#password cisco 
Router(config-line)#login 
Router(config-line)#

Configuring Enable Password

Enable password is required to restrict the access of privileged EXEC mode.

To configure the enable password of a CISCO router:

Enable password: 
Router>enable 
Router#config 
Router#configure terminal 
Router(config)#enable password cisco 
Router(config)#

To encrypt the enable password, use the following command:

Perform password encryption: 
Router>enable 
Router#config 
Router#configure terminal 
Router(config)#enable secret cisco 
Router(config)#

Encrypting All Passwords on Cisco Router

By default, all the passwords of a CISCO router are readable in clear text in the configuration file. This is a great security threat if someone reads it and configures or changes the router configuration. To protect from displaying the password, the service password-encryption command is used to encrypt the passwords. Service password-encryption is a global command and encrypt the passwords:

Router>enable 
Router#config 
Router#configure terminal 
Router(config)#service password-encryption 
Router(config)#

By following the above steps, you can easily configure CISCO router passwords.
