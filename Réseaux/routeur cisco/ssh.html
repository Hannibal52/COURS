Tutoriel Configuration Routeur Cisco

Configurer SSH sur un routeur Cisco

configure terminal

interface f0/0
#ip address w.x.y.z  w.x.y.z 
#no shutdown 
#do wr
#exit


ip domain-name votre-domaine.com
crypto key generate rsa general-keys modulus 2048
ip ssh version 2
username votre_utilisateur privilege 15 secret votre_mot_de_passe
line vty 0 15
login local
transport input ssh
exit
write memory
show ip interface brief show run | section line vty show run | include ssh

Configurer l'adresse de Loopback

configure terminal
interface Loopback 0
ip address x.x.x.x y.y.y.y
no shutdown
exit
write memory

Se connecter au routeur via SSH

ssh noro@x.x.x.x

en cas de problem

  • Assurez-vous que l'adresse IP du routeur est correcte.

  • Vérifiez que le serveur SSH est correctement configuré sur le routeur.

  • Assurez-vous que les lignes VTY sont correctement configurées pour accepter les connexions SSH.

	show ip interface brief
	 
	show run | section line vty
	 
	show run | include ssh
