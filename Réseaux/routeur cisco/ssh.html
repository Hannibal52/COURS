Tutoriel Configuration Routeur Cisco

Configurer SSH sur un routeur Cisco

configure terminal

interface f0/0
#ip address w.x.y.z  w.x.y.z 
#no shutdown 
#do wr
#exit


ip domain-name votre-domaine.com
crypto key generate rsa general-keys modulus 2048
ip ssh version 2
username votre_utilisateur privilege 15 secret votre_mot_de_passe
line vty 0 15
login local
transport input ssh
exit
write memory
show ip interface brief show run | section line vty show run | include ssh

Configurer l'adresse de Loopback

configure terminal
interface Loopback 0
ip address x.x.x.x y.y.y.y
no shutdown
exit
write memory

Se connecter au routeur via SSH

ssh noro@x.x.x.x

en cas de problem

  • Assurez-vous que l'adresse IP du routeur est correcte.

  • Vérifiez que le serveur SSH est correctement configuré sur le routeur.

  • Assurez-vous que les lignes VTY sont correctement configurées pour accepter les connexions SSH.

	show ip interface brief
	 
	show run | section line vty
	 
	show run | include ssh


Configuration Cisco

Commande Description
R1(config-line)#exec-timeout 0 60 Définit le temps d'inactivité avant que la session ne se termine
sh ssh Affiche les sessions en cours SSH
ip ssh authentication-retries ? Définit le nombre de tentatives autorisées pour une connexion SSH
ip ssh time-out 60 Définit le temps d'attente pour négocier une clé de session et une méthode de chiffrement avec un client SSH
Show ip ssh Affiche la configuration SSH
crypto key ? Modifie les clés SSH
security passwords min-length 10 Force une longueur minimale pour les mots de passe
login block-for 10 attempts 3 within 30 Bloque les connexions pendant 10 secondes si 3 tentatives de connexion échouent dans les 30 secondes
show login Affiche les paramètres de connexion
login quiet-mode access-class adminacl Bloque les messages de connexion pour les clients ne figurant pas sur la liste d'accès standard "adminacl"
enable secret level X ciscoX Définit un mot de passe secret et un niveau d'accès
privilege exec,configure,interface level X Définit les privilèges de niveau d'accès pour les utilisateurs
