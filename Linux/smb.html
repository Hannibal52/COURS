Installer et Configurer Samba sur Debian 11

Etape 1: Installer Samba

sudo apt-get update
sudo apt-get install samba

Etape 2: Creer le repertoire de partage

sudo mkdir /samba/share
sudo chown -R nobody:nogroup /samba/share
sudo chmod -R 0775 /samba/share

Etape 3: Configurer Samba

Editez le fichier de configuration Samba avec la commande suivante :

sudo nano /etc/samba/smb.conf

Ajoutez les lignes suivantes a la fin du fichier:

[Share]
path = /samba/share
browsable =yes
writable = yes
guest ok = yes
read only = no

Etape 4: Redemarrer Samba

sudo systemctl restart smbd

Etape 5: Verifier le partage Samba

smbclient -L localhost -U%

Etape 6: Acceder au partage depuis une machine Linux

Vous pouvez utiliser la commande smbclient pour acceder au partage Samba depuis une machine Linux.

smbclient //hostname_or_IP_address/Share

Etape 7: Acceder au partage depuis une machine Windows

Depuis une machine Windows, vous pouvez acceder au partage Samba en utilisant l'Explorateur de fichiers.

  1. Ouvrez l'Explorateur de fichiers.
  2. Dans la barre d'adresse, tapez le chemin d'acces au partage Samba, qui sera quelque chose comme 
    \\hostname_or_IP_address\Share
    .
