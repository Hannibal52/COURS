Azure key Vault

Azure Key Vault est un service qui vous permet de stocker et de gérer en toute sécurité des secrets, tels que des mots de passe, des clés d'API et des certificats. Pour récupérer un mot de passe stocké dans Azure Key Vault, vous pouvez utiliser le SDK Azure pour Python. Voici comment le faire :

  1. Installez le SDK Azure pour Python, si ce n'est pas déjà fait :

    
pip install azure-keyvault-secrets azure-identity

  2. Installez le SDK Azure pour Python, si ce n'est pas déjà fait :

    
from azure.identity import DefaultAzureCredential
from azure.keyvault.secrets import SecretClient
  3. Configurez les informations d'identification d'Azure. Vous pouvez le faire en configurant les variables d'environnement AZURE_TENANT_ID, AZURE_CLIENT_ID et AZURE_CLIENT_SECRET avec les informations d'identification appropriées pour votre compte Azure.

  4. Utilisez le code suivant pour récupérer un mot de passe stocké dans Azure Key Vault :

    
def get_password_from_key_vault(vault_url, secret_name):
    # Créez un objet DefaultAzureCredential
    credential = DefaultAzureCredential()
    
    # Créez un objet SecretClient en utilisant l'URL du coffre-fort et les informations d'identification
    secret_client = SecretClient(vault_url=vault_url, credential=credential)
    
    # Récupérez le mot de passe du coffre-fort en utilisant le nom du secret
    secret = secret_client.get_secret(secret_name)
    
    return secret.value

# Utilisation de la fonction pour récupérer un mot de passe
vault_url = "https://your-key-vault-name.vault.azure.net/"
secret_name = "your-secret-name"
password = get_password_from_key_vault(vault_url, secret_name)
print(f"Le mot de passe récupéré est : {password}")

Assurez-vous de remplacer your-key-vault-name par le nom de votre coffre-fort Azure Key Vault et your-secret-name par le nom du secret contenant le mot de passe que vous souhaitez récupérer.
