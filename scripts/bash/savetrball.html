Sauvegarde dans une archive tarball compressée

voici un exemple de script Bash pour sauvegarder les fichiers d'un répertoire spécifique dans une archive tarball compressée et la déplacer vers un répertoire de sauvegarde.


  #!/bin/bash

# Paramètres de configuration
SOURCE_DIR="/chemin/vers/le/repertoire/source"
BACKUP_DIR="/chemin/vers/le/repertoire/backup"
FILE_PREFIX="backup_"

# Récupérer la date actuelle
DATE=$(date +%Y%m%d_%H%M%S)

# Créer un nom de fichier pour l'archive de sauvegarde
BACKUP_FILE="${BACKUP_DIR}/${FILE_PREFIX}${DATE}.tar.gz"

# Créer l'archive tarball compressée
echo "Création de la sauvegarde : ${BACKUP_FILE}"
tar -czf "${BACKUP_FILE}" -C "${SOURCE_DIR}" .

# Vérifier si la création de l'archive a réussi
if [ $? -eq 0 ]; then
  echo "Sauvegarde créée avec succès : ${BACKUP_FILE}"
else
  echo "Échec de la création de la sauvegarde. Veuillez vérifier les erreurs ci-dessus."
fi


Assurez-vous de remplacer les valeurs des variables SOURCE_DIR et BACKUP_DIR par les chemins appropriés de vos répertoires source et de sauvegarde. Le script crée ensuite une archive tarball compressée des fichiers du répertoire source et la déplace vers le répertoire de sauvegarde avec un nom de fichier basé sur la date et l'heure actuelles. Vous pouvez exécuter ce script manuellement ou le planifier avec cron pour effectuer des sauvegardes régulières de vos fichiers.
