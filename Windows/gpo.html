Ajouter des GPO

Comment ajouter GPO Windows Server

Temps total : 5 minutes

  1. Ouvrir le gestionnaire de serveurs

    En premier lieu, ouvrez votre gestionnaire de serveurs.

  2. Gestion de stratégies de groupe

    Par la suite, aller sur outils en haut à droite et sur gestion de stratégies de groupe.

  3. Unités et stratégies de base

    Ici vous allez voir la forêt que vous avez pour votre active Directory, vous allez pouvoir l’étendre pour voir les stratégies de groupe et les unités de base.

  4. Modifier ou supprimer stratégie de base

    De base, vous avez la stratégie de base Default Group Policy qui sont des règles de sécurité de base, vous pouvez l’enlever si vous souhaitez mettre vos propres règles.

  5. Créer une unité d’organisation

    Par la suite, faites un clic droit sur votre forêt et faites ajouter une nouvelle unité d’organisation. Vous les nommez comme vous souhaitez et vous allez pouvoir mettre vos utilisateurs à l’intérieur.

  6. Nouvelle GPO

    Faites un clic droit sur l’unité d’organisation et faites ajouter une nouvelle stratégie de groupe GPO.

  7. Ajouter des règles

    Ici vous allez retrouver l’arborescence de toutes les règles qui sont disponibles à Windows. Vous allez pouvoir définir ces stratégies et tous les utilisateurs qui sont dans la même unité d’organisation que vos GPO auront, ces règles appliquées à cette unité d’organisation.

Alors oui, il va falloir mettre vos utilisateurs dans la même unité d’organisation ou sinon vous pouvez mettre votre GPO dans l’unité de base.

Comment trouver les GPO

Bon, maintenant que vous êtes en mode, je veux créer des GPO. Vous souhaitez sûrement savoir où trouver certaines stratégies de groupe avec votre Windows Server, car il y en a des milliers!

Alors je vais vous donner une petite stratégie pour comment trouver des stratégies de trouver des stratégies de groupe. Nous allons utiliser la fonction de filtre. 

Pour ajouter un filtre de recherche à un GPO:

Comment trouver GPO et utiliser le filtre Windows Server

Temps total : 5 minutes

  1. Stratégie à modifier

    Ouvrez la stratégie de groupe que vous voulez modifier.

  2. Arborscence à faire un filtre

    Sélectionnez l’arborescence parente que vous souhaitez faire le filtre de recherche, faites un clique droit dessus et filtre.

  3. Définir le filtre

    Définissez le filtre que vous souhaitez et vous allez pouvoir avoir les résultats de stratégie et règles qui que Windows vous propose.

Faire des GPO local sur Windows

À présent que dans vu comment ajouter des stratégies de groupe à notre Active Directory pour gérer les utilisateurs d’un domaine, nous allons voir comment ajouter des GPO local à votre ordinateur Windows si vous n’avez pas de serveur sous Windows Server ou si vous souhaitez modifier certaines fonctions de votre ordinateur. 

Créer GPO local Windows

Temps total : 5 minutes

  1. Menu exécuter -> GPEdit

    En premier lieu à appuyer sur la touche Windows+R pour ouvrir le menu exécuter.

    Écrivez, GPedit.msc et appuyez sur OK.

  2. Erreur ou autre

    Voilà les stratégies de groupe de votre ordinateur local. Si vous ne réussissez pas à accéder à cette fenêtre et qu’il y a une erreur, c’est parce que vous utilisez Windows home ou Windows famille et non Windows professionnel. Alors je vous invite à regarder ce tutoriel pour activer le GPEdit.msc sur Windows Home et Windows famille.

FAQ

pourquoi les gpo ne fonctionne pas ?

Si votre GPO ne fonctionne pas, vous pouvez vérifier si lorsque vous faites un clique droit dessus il y a « appliqué » de coché.

Vous pouvez aussi tapper sur l’ordinateur cible dans l’invite de commande: « gpupdate /force ».

Comment trouver une GPO ?

Si vous souhaitez trouver une GPO, vous pouvez faire un rechercher sur Google avec l’action à modifier + GPO. par exemple je souhaite bloquer le gestionnaire des tâches, j’écrirais sur Google « Bloquer Gestionnaire des tâches GPO » et vous allez trouver l’emplacement.
