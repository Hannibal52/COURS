Liste des raccourcis clavier Windows

    • ncpa.cpl : Ouvre la fenêtre de connexion réseau, qui permet de gérer les connexions réseau.
    • lusrmgr.msc : changer mot de passe
    • firewall.cpl : Ouvre la fenêtre de Pare-feu Windows, qui permet de gérer les règles de Pare-feu Windows.
    • appwiz.cpl : Ouvre la fenêtre des programmes et fonctionnalités, qui permet de désinstaller des programmes ou de modifier des fonctionnalités de Windows.
    • devmgmt.msc : Ouvre le Gestionnaire de périphériques, qui permet de gérer les périphériques matériels de l'ordinateur.
    • diskmgmt.msc : Ouvre l'outil de gestion des disques, qui permet de gérer les disques durs, les partitions et les volumes.
    • compmgmt.msc : Ouvre la console de gestion de l'ordinateur, qui permet de gérer différents aspects de l'ordinateur, tels que les utilisateurs et les groupes, les services, les événements, etc.
    • msinfo32.exe : Ouvre l'outil d'informations système, qui fournit des informations détaillées sur la configuration matérielle et logicielle de l'ordinateur.
    • msconfig.exe : Ouvre l'outil de configuration du système, qui permet de modifier les paramètres de démarrage, les services, les processus, etc.
    • perfmon.msc : Ouvre l'outil de surveillance de la performance, qui permet de surveiller l'utilisation des ressources système.
    • secpol.msc : Ouvre la console de stratégie de sécurité locale, qui permet de gérer les paramètres de sécurité de l'ordinateur.
    • services.msc : Ouvre la console des services, qui permet de gérer les services système et les services de démarrage automatique.
    • taskmgr.exe : Ouvre le Gestionnaire des tâches, qui permet de surveiller et de gérer les processus en cours d'exécution sur l'ordinateur.
    • control.exe : Ouvre le panneau de configuration, qui permet de gérer les paramètres système de l'ordinateur.
    • regedit.exe : Ouvre l'éditeur du Registre, qui permet de gérer la base de données du Registre de Windows.
    • gpedit.msc : Ouvre l'Éditeur de stratégie de groupe, qui permet de gérer les paramètres de sécurité et de configuration de l'ordinateur pour un groupe ou un utilisateur spécifique.
