Tutoriel : Docker et Kubernetes sur Windows

Remarque : Docker et Minikube ont tous deux des exigences système spécifiques. Par exemple, vous devez disposer de la virtualisation activée dans le BIOS et d'une version de Windows 10 64 bits.

Étape 1: Installer Docker

  1. Allez sur le site officiel de Docker à l'adresse https://www.docker.com/products/docker-desktop et cliquez sur "Get Docker Desktop For Windows".
  2. Une fois le fichier d'installation téléchargé, ouvrez-le pour lancer l'installation.
  3. Suivez les instructions à l'écran pour terminer l'installation.

Étape 2: Installer Minikube

  1. Ouvrez PowerShell en mode administrateur.
  2. Utilisez la commande suivante pour télécharger Minikube : 
    Invoke-WebRequest -Uri https://storage.googleapis.com/minikube/releases/latest/minikube-windows-amd64.exe -OutFile minikube.exe
  3. Déplacez le fichier minikube.exe dans un répertoire qui est dans votre PATH, par exemple : 
    Move-Item -Path .\minikube.exe -Destination C:\Windows\System32

Étape 3: Démarrer Minikube

    Note : Dans la console PowerShell, tapez la commande suivante pour démarrer Minikube avec le runtime de conteneur cri-o : 
  1. minikube start --container-runtime cri.o
